Arquitetura de Agentes de IA e Engenharia de Prompt

Visão Geral

Este sistema é um assistente de e-commerce que transcende a busca por palavras-chave tradicional. Ele utiliza NLP (Natural Language Processing) para decodificar a intenção semântica do usuário, permitindo buscas complexas e contextualizadas.

Agent Engineering

  • Agente de Intenção: Um estágio via LLM analisa a query para identificar categorias, extrair orçamentos e filtrar ruído.
  • RAG (Retrieval-Augmented Generation): O assistente consulta um banco de dados real antes de gerar recomendações, evitando alucinações.
